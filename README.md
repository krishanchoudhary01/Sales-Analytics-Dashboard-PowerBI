# Super Store Sales Dashboard

## Project Overview
This project is an interactive Sales Analytics Dashboard built using Python Pandas and Power BI.

The project focuses on:
- Data Cleaning
- Data Analysis
- Data Visualization
- Business Insights
- Interactive Dashboard Creation

---

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Power BI
- DAX

---

# Learning

Incorporated Python Pandas for data cleaning, preprocessing, and exploratory data analysis. Applied data analysis techniques, including time series analysis, KPI tracking, sales analysis, and interactive dashboard creation using Power BI to generate valuable business insights and support data-driven decision making.

# Key Business Insights

- Technology category generated highest sales
- COD was the most used payment mode
- Phones were the top-selling sub-category
- Some regions showed high sales but lower profit margins
- Monthly sales trends identified seasonal performance patterns
